COVID-19 Pneumonia Detection Using Residual Attention Network (Image Classificaton)

Tools used: Python, Tensorflow, Scikit-learn, Numpy, Pandas, Matplotlib, Seaborn

Background of the project

Covid-19 is a respiratory infection that mainly affects the human lungs, which has been declared as a global pandemic that affected the globe. Covid-19 and Pneumonia mainly damages human lungs. Compared to people who have pneumonia, those affected by COVID-19 have significantly worse lung damage. Even most people recover from pneumonia without any lasting lung damage. That’s why the deep learning paradigm has been applied vastly to investigate radiographic images such as Chest X-Rays (CXR) to detecting covid-19 and classifying it from the viral pneumonia. In this project, we are proposing a method to detect COVID-19 and classifying it from Pneumonia with chest X-rays using a deep learning.

Methodology

In this method, we will collect chest X-rays from the patients of three categories. These categories are COVID-19, Pneumonia and Normal(Healthy) chest X-rays. We will take the images of these X-rays and create a dataset where all three category of X-rays will exist. Then the dataset will be preprocessed by splitting the dataset into three parts, training, validation and testing. After that we will apply data augmentation technique to increase the amount of data that we are working with by adding slightly modification. After preprocessing the dataset, we will train our model(Residual Attention Network) with the training data. After training the model we will evaluate our model with validation data. Finally our model will be ready to predict the class of any given chest x-ray and classify it among three class, COVID-19, Pneumonia and Normal.

Dataset

We have used chest X-rays of three classes: COVID-19, Pneumonia, Normal.This is a combined curated dataset of COVID-19 Chest X-ray images obtained by collating 15 publically available datasets . The present dataset contains 1207 COVID-19 X-Rays, 3270 Normal X-Rays and 3556 Pneumonia X-Rays. We have converted all the images into 256x256 (height x width) to make the training faster.

Model Summary

Residual Attention Network that we used is constructed by stacking up Attention module and Residual module onto one another. Our model had 10.43 million parameters and among them 10.40 million parameters were trainable and almost 26.8k parameters were non-trainable

Result (Accuracy and loss)

The accuracy and the loss of the model is verified by the validation data. The model predicts accuracy and loss for the validation data. Accuracy vs Validation Accuracy and Loss vs Validation loss represents how well a model is trained. Training parameters and validation parameters will be close if the model or neural network is being trained properly. The closer these parameter are the model is more well-trained. In our model, we can see that both training and validation parameter(accuracy and loss) follow each other quite well.

Result (Confusion Matrix)

A multi-class supervised learning problem can be visualized using the confusion matrix by building a matrix. In the matrix, each line depicts the occurrences in a predicted class, and each column shows the instances in the actual class. The Confusion Matrix for our classification problem has three classes and it is a 3*3 matrix that has nine parameters.From the confusion matrix, we can see that true positive value for Normal, Pneumonia and Covid is 0.92, 0.98 and 0.98 accordingly.

Result (F1 Score, Precision, Recall)

By merging the recall and precision evaluation matrices into a single performance measure evaluation matrices, the F1 score evaluation matrix is used to assess the performance of a classifier.Here, we have obtained F1 score of 0.95 for Normal, 0.96 for Pneumonia and 0.98 for COVID-19 class. Among the three classes the model is performing better for COVID-19 detection. We have achieved 0.96 overall F1 score for all three classes where 1.00 is the ideal scenario. This overall F1 score indicates the excellent performance of our model for this particular problem.

Result (ROC Curve)

A statistic for gauging a classifier's effectiveness is the roc area. This is achieved by plotting the genuine positive rate versus the false positive rate. The accuracy of the classifier is then evaluated using the area under the generated roc curve. The area is nearer to 1 the better the classifier. Covid detection and classification is a multi-class problem and that’s why we can see one curve for each class. We get the AUC score 0.99 for Pneumonia, 0.99 for Normal and 1.00 for COVID-19. Here, the idea score for each class is 1.00. So from the curve, we can see that the true positive rate for each class is really good so the performance of the model is excellent.
